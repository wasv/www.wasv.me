Useful Resources

z80

  • rc2014.co.uk Incredibly well designed and documented modular z80. The basis of my own z80 computer.
  • z80pack A z80 emulator that I used for playing with z80 assembly.
  • Sharp PC-E220 The z80 pocket computer I really, really want.

TI-83

  • WikiTI Extremely useful reference for writing TI-83+ firmware or apps.
  • KnightOS Custom firmware for the TI-8x.
  • TiLP Linking software for copying files to/from TI calculators.

OpenGL

  • docs.gl Useful reference for OpenGL functions and compatibility.
  • opengl-tutorial.org Tutorial 7 on model loading is particularly useful. I use a modified version of the code in most of my OpenGL projects.
  • open.gl The first OpenGL tutorial I used. Went from a basic toolchain to an animated textured quad over the course of a 24 hour hackathon.
  • learnopengl.com A more advanced OpenGL tutorial.
